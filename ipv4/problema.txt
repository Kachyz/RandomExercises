Problema
  Crear una función para validar que un valor recibido es una IPv4 válida

Input
  IP a validar

Output
  Boolean, true o false de acuerdo a la validez de la IP recibida

Ejemplo
  Input - 192.168.1.1
  Output - true

  Input - 192.168.280.0
  Output - false


Adicional
  Integrar pruebas unitarias sencillas para validar el funcionamiento del método generado
